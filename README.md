# Projeto Full-Stack

## Sobre o Projeto

Este mini projeto foi desenvolvido com o apoio de aulas no YouTube e tem como objetivo principal o aprimoramento prático em **React.js** e **Node.js**, por meio da construção de uma **API REST** responsável por integrar o Front-End com o Back-End.

O foco do projeto é consolidar conceitos como:

- Criação de APIs REST
- Consumo de APIs no Front-End
- Integração com banco de dados
- Organização de código e boas práticas

---

## Objetivo

Desenvolver uma aplicação Full-Stack funcional, aplicando na prática os conhecimentos adquiridos em:

- Desenvolvimento Front-End com React
- Desenvolvimento Back-End com Node.js
- Persistência de dados com MariaDB

---

## Padrão de Commits

Para manter o histórico de versões organizado, foi adotado o seguinte padrão de commits:

| Tipo de alteração | Prefixo |
| ----------------- | ------- |
| Desenvolvimento   | `[dev]` |
| Documentação      | `[doc]` |
| Banco de Dados    | `[db]`  |

### Exemplos

#### bash

[dev] Criação da rota de usuários
[db] Ajuste na estrutura da tabela users
[doc] Atualização do README
